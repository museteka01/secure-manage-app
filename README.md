# Secure Manage App

A containerized Flask API with automated CI/CD using GitHub Actions.

## Tech Stack
- Python / Flask
- Terraform
- Docker
- GitHub Actions
- pytest, flake8, bandit

## Features
- API endpoints for secure resource management
- Automated testing
- Linting and security scanning in CI
