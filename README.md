# Secure Manage App

Secure Manage App is a backend service built with **FastAPI** that demonstrates
secure authentication, database integration, containerization with Docker,
and infrastructure concepts commonly used in DevOps / DevSecOps workflows.

This project is intended as a learning and portfolio project, focusing on
clarity, security practices, and real-world structure.

---

## Tech Stack

- Python 3.10+
- FastAPI
- PostgreSQL
- SQLAlchemy
- Docker
- Terraform (infrastructure)
- CI/CD pipelines (ci-cd folder)
